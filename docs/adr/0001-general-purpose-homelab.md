# General-purpose homelab, not single-service appliance

The infrastructure is scoped as a general-purpose homelab that will host multiple services (Immich, Nextcloud, status page, and future additions), not a dedicated Immich appliance. This means the repo, Ansible roles, networking, and disk layout are designed for multi-service growth from the start. The alternative -- scoping narrowly to Immich -- would have been simpler initially but would require restructuring when the next service is added.
